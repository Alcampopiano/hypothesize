Hypothesize: robust statistics in Python

Hypothesize is a robust statistics library for Python based on Rand R. Wilcox's R package WRS. With Hypothesize you can compare groups and measure associations using methods that outperform traditional statistcal approaches in terms of power and accuracy.

For more information on robust methods please see Wilcox's book Introduction to Robust Estimation and Hypothesis Testing.
